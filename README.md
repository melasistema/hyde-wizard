### ğŸš€ Installation

1. **Clone the repository:**
```bash
git clone git@github.com:melasistema/hyde-wizard.git
```

2. **Navigate into the project folder:**
```bash
cd hyde-wizard
```

3. **Install dependencies:**

```bash
composer install

```   

4. **Compile assets and serve the site:**

